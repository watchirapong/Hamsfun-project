# Hamsfun Project

This is a [Next.js](https://nextjs.org/) project bootstrapped with TypeScript, Tailwind CSS, and ESLint.

## Getting Started

First, make sure you have Node.js installed (version 18 or higher). Then install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `src/app/` - App Router directory containing your pages and layouts
- `src/app/layout.tsx` - Root layout component
- `src/app/page.tsx` - Home page
- `src/app/globals.css` - Global styles with Tailwind CSS
